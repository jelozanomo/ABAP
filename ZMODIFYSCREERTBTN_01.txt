REPORT ZMODIFYSCREERTBTN_01.

SELECTION-SCREEN BEGIN OF BLOCK BLA WITH FRAME TITLE A1TITLE.

  SELECTION-SCREEN BEGIN OF BLOCK BLB WITH FRAME TITLE B1TITLE.

    PARAMETERS: R1 RADIOBUTTON GROUP GR1 USER-COMMAND CMD DEFAULT 'X',
                R2 RADIOBUTTON GROUP GR1,
                R3 RADIOBUTTON GROUP GR1.

    SELECTION-SCREEN BEGIN OF BLOCK BLB1 WITH FRAME.

      PARAMETERS: RP1 RADIOBUTTON GROUP GR2 MODIF ID CD,
                  RP2 RADIOBUTTON GROUP GR2 MODIF ID CD,
                  RP3 RADIOBUTTON GROUP GR2 MODIF ID CD,
                  RP4 RADIOBUTTON GROUP GR2 MODIF ID CD.
    SELECTION-SCREEN END OF BLOCK BLB1.

    SELECTION-SCREEN BEGIN OF BLOCK BLB11 WITH FRAME.
      PARAMETERS:  RO1 RADIOBUTTON GROUP GR3 MODIF ID CD1,
                   RO2 RADIOBUTTON GROUP GR3 MODIF ID CD1.

    SELECTION-SCREEN END OF BLOCK BLB11.

  SELECTION-SCREEN END OF BLOCK BLB.


  SELECTION-SCREEN BEGIN OF BLOCK BLC WITH FRAME TITLE C1TITLE.
    PARAMETERS: P_CARID1 LIKE SCARR-CARRID,
                P_CARID2 LIKE SCARR-CARRID .

  

  SELECTION-SCREEN END OF BLOCK BLC.

SELECTION-SCREEN END OF BLOCK BLA.
INITIALIZATION.
  PERFORM TEXT.

AT SELECTION-SCREEN OUTPUT.

  IF R1 EQ 'X'.
    LOOP AT SCREEN.
      IF SCREEN-GROUP1 EQ 'CD'.
        SCREEN-ACTIVE = 1.
     MODIFY SCREEN.
     ENDIF.
    ENDLOOP.
  ELSE.
    LOOP AT SCREEN.
      IF SCREEN-GROUP1 EQ 'CD'.
        SCREEN-ACTIVE = 0.
     MODIFY SCREEN.
     ENDIF.
    ENDLOOP.
  ENDIF.

  IF R2 EQ 'X'.
    LOOP AT SCREEN.
      IF SCREEN-GROUP1 EQ 'CD1'.
        SCREEN-ACTIVE = 1.
     MODIFY SCREEN.
     ENDIF.
    ENDLOOP.
  ELSE.
    LOOP AT SCREEN.
      IF SCREEN-GROUP1 EQ 'CD1'.
        SCREEN-ACTIVE = 0.
     MODIFY SCREEN.
     ENDIF.
    ENDLOOP.
  ENDIF.

FORM TEXT.
  A1TITLE = 'Pantalla de creación - Cliente/Deudor'.
  B1TITLE = 'El interlocutor comercial se crear como?'.
  C1TITLE = 'Información del cliente'.
  D1TITLE = 'Dirección Calle'.
ENDFORM.
